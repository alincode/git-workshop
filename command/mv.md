# git mv 指令

### 常用範例

| 範例                | 說明  |
|-------------------|-----|
| git mv README.txt README2.txt |     |

### 語法結構

```
usage: git mv [<options>] <source>... <destination>

    -v, --verbose         be verbose
    -n, --dry-run         dry run
    -f, --force           force move/rename even if target exists
    -k                    skip move/rename errors
```
